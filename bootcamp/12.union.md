# Union

In C, a **union** is a special data type that allows you to store different data types in the same memory location. Unlike a `struct`, where each member has its own memory space, members of a `union` share the same memory. This means only one member can hold a value at any given time.

### Defining a Union
```c
union union_name {
    data_type variable_name;
    data_type variable_name;
};
```

* **union**: A keyword to define a union.
* **union_name**: The name of the union.
* **data_type variable_name**: Create a member of the union, which can be any data type.

Example:
```c
union Data {
    int i; // 4 bytes
    float f; // 4 bytes
    char str[20]; // 20 bytes
};
```

>[!Note]
> All members of the union share the same memory space. The size of the union is determined by the size of its largest member; the size of `union Data` will be **20 bytes** (the size of `str`).



### Example
To use a union, declare it within the `main()` function, just like a `struct`.

```c
#include <stdio.h>

union Data {
    int i;
    float f;
    char str[20];
};

int main() {
    union Data variable;

    variable.i = 42;  // Assign integer
    printf("Integer: %d\n", variable.i);

    variable.f = 3.14;  // Assign float (overwrites previous value)
    printf("Float: %.2f\n", variable.f);
    printf("Integer: %d\n", variable.i); // Accessing `variable.i` here will result in an undefined behavior value because the memory is overwritten.

    sprintf(variable.str, "Hello");  // Assign string (overwrites previous value)
    printf("String: %s\n", variable.str);

    return 0;
}
```

Output:
```
Integer: 42
Float: 3.14
Integer: 1078523331
String: Hello
```

>[!Note]
>Since all members share the same memory, assigning a new value to one member will overwrite the previous value.
